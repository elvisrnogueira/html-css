<-- O que é Git e GitHub -->

1- Repositório Local:
    1.1- Git - Gerenciador de versionamento locais
    1.2- Mantem versões do software dentro do seu computador
    1.3- Se perder o computador, você perde os arquivos
2- Repositório Remoto:
    2.1- GitHub - Gerenciador de versionamento remoto
    2.2- Faz um backup das versões do seu software na nuvem
    2.3- Pode ser usado para trabalhar em equipes
3- Quando tiver experiência, assistir o curso de Git e GitHub para se aprofundar

<-- Instalando Git e GitHub no PC -->

1- site para download do Git: https://git-scm.com/
2- site para download do GitHub: https://desktop.github.com/download/

<-- Primeiro repositório git e GitHub -->

1- Como criar o seu repositório Git:
    1.1- Abra o GitHub desktop
    1.2- Clique em file e New repository (CTRL + N)
    1.3- Crie um nome para o repositório
    1.4- De uma descrição do repositório
    1.5- Coloque o caminho de onde estão seus arquivos
    1.6- Marque para inicializar com um README
    1.7- Deixe o Git Ignore como none
    1.8- Recomendado usar a licença: MIT License
    1.9- Depois clique em Create Repository
2- Como publicar seu repositório no GitHub:
    2.1- Abra o GitHub desktop
    2.2- Clique em Publish repository
    2.3- Desmarque a opção de deixar o repositório privado - caso contrário ele estará privado de outros usuários

<-- Gerenciando Repositório GitHub -->

1- Como atualizar o seu repositório:
    1.1- Faça a alteração no arquivo de preferência
    1.2- Abra o GitHub desktop
    1.3- Adicione um Título a sua alteração (descrevendo o que foi feito)
    1.4- Realize o Commit
    1.5- Realize o Push
2- Como apagar seu repositório Git:
    2.1- Abra o GitHub desktop
    2.2- Selecione o repositório que deseja apagar
    2.3- Clique em Repository e depois em Remove
3- Como apagar seu repositório do GitHub:
    3.1- Abra o GitHub no navegador
    3.2- Selecione o repositório que deseja apagar
    3.3- Clique na opção Settings
    3.4- Clique em Delete this repository

<-- Hospedando site gratuitamente com GitHub Pages -->

1- Para hospedar gratuitamente no GitHub você deve:
    1.1- Abrir o GitHub no navegador
    1.2- Selecionar o repositório de preferência
    1.3- Abrir o Settings
    1.4- Abrir Pages
    1.5- Em branch você escolhe a opção main depois /(root) e depois salvar para criar sua URL
2- Dica: acesse o site qr-code-generator.com para gerar um QR Code para seu site

<-- Clonando Repositório GitHub -->

1- Abra o GitHub no navegador
2- Selecione o repositório de preferência
3- Clique em <> Code 
4- Depois clique em Open with GitHub Desktop para abrir diretamente no GitHub Desktop ou Download ZIP para baixar o ZIP do repositório

<-- GitHub em vários PCs -->

1- Depois que tiver logado no outro PC, ter editado o repositório do jeito que quiser, e ter feito um push no GitHub,
   lembre-se de fazer log out e remover o repositório do PC

<-- Colocando uma imagem de fundo no seu site -->

1- Para colocar uma imagem de fundo, você deve usar a declaração:
    1.1- background-image: url('caminho do arquivo de imagem');
2- Para fazer com que a imagem de fundo repita várias vezes, você deve usar a declaração:
    2.1- background-repeat: (valor);
    2.2- Os valores podem ser:
        2.2.1- repeat : para repetir a imagem por todo o site
        2.2.2- no-repeat : para termos apenas uma única imagem
        2.2.3- repeat-x : para repetir apenas para o eixo x
        2.2.3- repeat-y : para repetir apenas para o eixo y
    2.3- A imagem de fundo sempre partirá do canto superior esquerdo
3- A imagem de fundo sempre repetirá no tamanho padrão, a não ser que usemos a declaração:
    3.1- background-size: (valor);
    3.2- Os valores podem ser:
        3.2.1- contain : faz com que a imagem seja 100% exibida (pode criar barras pretas ao redor da imagem)
        3.2.2- cover : faz com que a imagem cubra a tela inteira (pode cortar as laterais da imagem para preencher a tela)
        3.2.3- auto : faz com que o navegador arrume o tamanho de maneira automatica (valor padrão)
        3.2.4- px/ % : altera o tamanho da imagem em pixel ou porcentagem da tela
4- Para configurar a posição da imagem de fundo, você deve usar a declaração:
    4.1- background-position: (valor);
    4.2- Os valores podem ser:
        4.2.1- top
        4.2.2- left
        4.2.3- center
        4.2.4- bottom
        4.2.5- right
    4.3- Podemos tambem combinar esses valores:
        4.3.1- left top
        4.3.2- center top
        4.3.3- right top
        4.3.4- left center
        4.3.5- center center
        4.3.6- right center
        4.3.7- left bottom
        4.3.8- center bottom
        4.3.9- right bottom
5- O vh significa Viewport Height (Altura do Viewport) é uma unidade de medida relativa que representa 1% da altura da janela visível do navegador
6- Para facilitar no entendimento do background-position, alteramos a medida do height com vh

<-- background-attachment e shorthand -->

1- A propriedade background-attachment no CSS define se a imagem de fundo de um elemento rola junto com o conteúdo da página ou se permanece fixa na tela enquanto você navega
2- Como declarar:
    2.1- background-attachment: (valor);
3- Os valores podem ser:
    3.1- Scroll : Para o fundo descer ou subir junto com o conteúdo
    3.2- Fixed : Para o fundo ficar fixo em um unico lugar e não mexer com o conteúdo
4- Podemos usar uma shorthand para declarar tudo que envolva o background
5- Como declarar:
    5.1- background: (color) (image) (position) (repeat) [size] (attachment);
    5.2- Embora na documentação esteja dizendo que podemos usar o size no shorthand, ele não está funcionando, temos que declara-lo separadamente
    5.3- Ex: background: black url("images/wallpaper002.jpg") center center no-repeat fixed;
             background-size: cover;
