<-- Harmonia das cores -->  

1- Cores primárias: Amarelo, Vermelho, Azul
2- Cores secundárias: Laranja, Violeta, Verde
3- Cores terciárias: Amarelo-esverdeado, Amarelo-alaranjado, Vermelho-alaranjado, Vermelho-arroxeado, Azul-arroxeado, Azul-esverdeado
4- Cores terciárias são misturas das cores primárias com as secundárias
5- Escreve-se primeiro o nome da cor primária e depois o da secundária para se criar o nome da terciária
6- As cores em tons pastéis tem cores terciárias 
7- Temperaturas de cores: cores frias e cores quentes
8- Cores frias: do Amarelo-esverdeado ao Violeta
9- Cores quentes: do Amarelo ao Vermelho-arroxeado
10- Geralmente a paleta de cores deve ter de 3 a 5 cores no máximo - desconsiderar branco e preto (SEMPRE VAI TER)
11- Dica: usar a cor primaria da logo do seu cliente para se basear na paleta de cor que terá o site/app
12- Cores Complementares: Caso queira usar uma cor que contraste com a outra, buscar cores opostas no circulo cromâtico
13- Cores Análogas: Caso queira usar cores similares, buscar cores ao lado no circulo cromâtico
14- Cores Análogas e uma complementar: Buscar cores ao lado no circulo cromâtico e uma oposta a cor principal
15- Cores Análogas Relacionadas: Escolher duas cores visinhas principais, pular uma cor, e escolher a próxima no circulo cromâtico
16- Cores Intercaladas: Escolher uma cor principal, pular uma, escolher a outra, pular outra, e escolher mais uma cor 
17- Exemplos de cores intercaladas:
    17.1- Cores Triádicas: Escolher uma cor e pular 3 cores, formando um triangulo equilatero
    17.2- Cores em Quadrado: Escolher uma cor e pular 2 cores, formando um quadrado
    17.3- Cores Tetrádicas: Escolher duas cores e as cores complementares a elas, formando um retângulo
18- Monocromia: Escolher uma única cor e mudar o contraste e a luminosidade dela

<-- Como criar um degradê com CSS -->

1- background-image: linear-gradient(to (position), color, color);
2- essa propriedade faz com que possamos selecionar mais de uma cor para fazermos um degradê entre elas na direção que quiser no fundo
3- substitua o position por top (cima), right (direita), bottom (baixo), left (esquerda)
4- podemos usar também a angulação exemplo: (90deg) - 90° graus
5- podemos também controlar a porcentagem de quanto queremos que mostre as cores, ex: #3198E2 20%, #6D59C0 40%,
6- background-image: radial-gradient(to circle, color, color);
7- essa propriedade faz com que possamos selecionar mais de uma cor para fazermos um degradê entre elas em circulo no fundo
8- background-attachment: fixed; - faz com que o fundo fique fixo
9- dica: não exagere nas cores - degradês de 3 a 5 cores
10- Ex: background-image: liner-gradient(to right, #3198E2 20%, #6D59C0 40%);
11- Ex: background-image: radial-gradient(to circle, #3198E2 20%, #6D59C0 40%);

<-- Tipografia: Parte 1 -->

1- Monge Copista/ Amanuense = Pessoa que copiava tudo a mão
2- Em 1450, Johannes Gutenberg (Pai da imprensa) inventou a prensa mecânica de tipo móveis
3- Os tipos móveis (letras), podiam ser de metal, madeira, argila e rolha cortisa
4- A tipografia veio para melhorar a visibilidade e legibilidade das letras
5- Tipografia = Estudo de como escrever coisas no papel

<-- Tipografia: Parte 2 - Anatomia do tipo -->

1- Para criar uma fonte nova, primeiro é desenhado o x minúsculo para medir a altutra base de todas as letras minúsculo (altura-x)
2- Altura do vazamento das letras minúsculas para cima se chama (ascendente)
3- Altura do vazamento das letras minúsculas para baixo se chama (descendente)
4- Altura das letras maiúsculas se chama (altura das maiúsculas)
5- A soma de todas as alturas nós chamamos de (corpo)
6- Anatomicos Geométricos:
    6.1- O prolongamento de algumas letras se chama (serifa) - serve para facilitar a leitura
    6.2- As Hastes são os traços verticais das letras
    6.3- Os filetes são o que une duas haste ou um ou mais arcos
    6.4- Os Arcos são os arcos (parte arredondada) das letras
    6.5- O Esporão serve para visualmente criar a base da letra
    6.6- O Vértice é a parte pontuda de uma letra
    6.7- O Terminal é quando eu tenho um elemento que não esta preso em outra letra
7- Anatomicos Partes do Corpo:
    7.1- Elemento que sai de uma haste e te leva para cima se chama braço
    7.2- Elemento que sai de uma haste ou de um braço e te leva para baixo se chama perna
    7.3- Elemento que fica no final de uma perna se chama pé (base que segura a fonte)
    7.4- Elemento que faz curva em uma letra como a letra "S" se chama espinha
    7.5- Quando temos dois arcos juntos nós chamamos de barriga
    7.6- Buraco na letra se chama olho
    7.7- Extensão na parte de cima da letra se chama orelha
    7.8- A parte que parte da descendente da letra se chama cauda
8- Glifos são todas as letras/ caractéres de uma unica fonte
9- O conjunto de todos os glifos nós chamamos de Fonte
10- Familia tipográfica: light, normal, semibold, bold, extrabold
11- Todas as fontes tem "normal" mas nem todas te familia tipográfica
12- Categorias de Fontes:
    12.1- Fonte Serifadas: com serifas
    12.2- Fonte sans-serif: sem serifas
    12.3- Fonte monoespaçadas: com ou sem serifas, todas tem a mesma largura
    12.4- Fonte handwriting/ scripitada: simula a letras feitas a mão
    12.5- Fonte display/ comemorativas: não se aplicam as demais regras

<-- Família de fonte com CSS -->

1- font-family: Arial, Helvetica, sans-serif;
2- usamos essa declaração para escolher a fonte que queremos

<-- Tamanho de fonte e suas medidas -->

1- Medidas Absolutas: cm, mm, px, pt, pc
2- Medidas Relativas: em, ex, rem, vw, vh, %
3- Recomendado pela W3C usar apenas as medias de (px) e (em)
4- 16px é o padrão sem colocar o font-size
5- o (em) mutiplica o tamanho natural da fonte (16px), ex: (2em) = (32px)
6- o (rem) mutiplica o tamanho principal da font-family declarada

<-- Usando o id com CSS -->

1- Em HTML é id = em CSS é # (só pode um elemento)
2- EM HTML é class = em CSS é . (pode vários elementos)

<-- As diferenças entre id e class -->

1- De acordo com o W3C, dentro de um mesmo documento HTML só se pode ter um elemento com id
2- id deve ser único e class pode ser múltiplo
3- Nomeie suas class de acordo com suas funções
4- id sobrepõe os valores de class
5- podemos ter mais de uma class em um único elemento - separamos as classes por espaço
6- podemos ter um id e uma class em um único elemento

<-- Pseudo-classes em CSS -->

1- display: inline-block; usamos para colocar tudo na mesma linha
2- Em CSS uma pseudo-classe é representada por :
3- as pseudo-classes precisam estar relacionadas a um elemento ou uma classe - são relacionadas ao estado de um elemento
4- :hover - quando passar com o mouse o estado do elemento muda
5- :visited - usado para alterar o estado dos elementos que já foram visitados como links
6- :active - usado para alterar o estado dos elementos que forem ativados / clicados
7- usamos > para indicar que queremos mudar um elemento que é herdeiro de outro - EX: div > p 
8- usamos a declaração display com o valor none para que não apareça o conteudo, e usamos o valor block para que apareça
9- Ex: display: none; - display: block;

<-- Pseudo-elementos em CSS -->

1- os pseudo-elementos mexem diretamente com o conteúdo do elemento
2- em CSS um pseudo-elemento é representado por ::
3- usamos ::after para adicionar um pseudo elemento depois de um elemento (after = depois)
4- usamos ::before para adicionar um pseudo elemento antes de um elemento (before = depois)
5- usamos a declaração content: para adicionar um conteúdo em um elemento
6- Ex: a::after {
            content: " [link]";
        }
    6.1- O exemplo anterior está adicionando uma pseudo-classe depois do elemento com o conteudo de valor [link]
7- Ex: a::before {
            content: "[link]";
        }
    7.1- O exemplo anterior está adicionando uma pseudo-classe antes do link com o conteudo de valor [link]

<-- Modelo de Caixas: primeiros passos -->

1- Tudo que é exibido no HTML são exibidos em formato de caixas 
2- Aninhamento é o nome dado quando existe uma caixa dentro da outra
3- Anatomia de uma caixa:
    3.1- height = altura do conteúdo
    3.2- width - largura do conteúdo
    3.3- padding - acolchoamento / preenchimento - espaço entre o conteúdo e a borda
    3.4- border - borda do elemento
    3.5- outline - contorno / trassado - fora da borda e dentro da margem
    3.6- margin - margem - espaço externo
4- Tipos de Caixa:
    4.1- box-level: Cria um elemento que ocupa toda a largura disponível do container (normalmente a tela)
e sempre começa em uma nova linha, empurrando o próximo elemento para baixo
    4.2- Ex: <div>, <h1> - <h6>, <p>, <main>, <header>, <nav>, <article>, <aside>, <footer>, <form>, <video>
    4.3- inline-level: Cria um elemento que ocupa apenas o espaço do seu conteúdo e fica na mesma linha que outros elementos,
sem quebrar linha automaticamente
    4.4- Ex: <span>, <a>, <code>, <small>, <strong>, <em>, <sup> - <sub>, <label>, <button>, <input>, <select>

<-- Modelo de caixas (parte 1) -->

1- user agent = seu navegador
2- display: block; = box-level
3- display: inline-block; = inline-level
4- border-style = mexe com o estilo da borda 
5- value do border-style: solid = solida, dashed = tracejada, dotted = pontilhada, double = linha dupla
6- border-width = mexe na largura da borda
7- border-color = mexe na cor da borda
8- ordem do modelo de caixas = top, right, bottom, left

<-- Modelo de caixas (parte 2) -->

1- podemos altera a margin quando um elemento está sobrepondo o box
2- margin: auto; - com que o box fique centrrlizado horizontalmente
3- a sintaxe do outline é a mesma do border
4- lembrar sempre de usar o shorthand para facilitar a leitura do código
5- border: (width) (style) (color);
6- padding: (top) (right) (bottom) (left); padding: (top - bottom) (right - left); padding: (total);
7- margin: (top) (right) (bottom) (left); margin: (top - bottom) (right - left); margin: (total);
8- OBS: podemos colocar também no lugar do right e left o valor auto, para que possamos centralizar o box horizontalmente
9- podemos transformar um inline-level em um box-level e vice-versa
10- display: inline; e display: inline-block; é usado para transformar um box-level em inline-level OBS: não é usado height e width no display: inline;
11- display: block; é usado para transformar um inline-level em box-level

<-- Sombra nas Caixas -->

1- sintaxe: box-shadow: (deslocamento horizontal) (deslocamento vertical) (blur) (espalhamento) (cor);
2- ex: box-shadow: 1px 1px 1px black;
3- dentro do DevTools do Chrome ao digitar a sintaxe padrão do box-shadow com todos os valores 0 no seletor de preferencia, podemos usar uma ferramenta visual para nos orientar na hora de montar nossa sombra
4- recomendado usar sombras pretas e apenas diminuir ou aumentar sua transparência
5- podemos ter uma sombra tanto outset = externa quanto inset = interna
6- para termos uma sombra inset devemos colocar o valor (inset) antes dos outros valores
7- ex: box-shadow: inset 2px 1px 14px 6px black;
8- text-shadow cria sombras no texto

<-- Caixas com vértices arredondados -->

1- sintaxe: border-radius: (valor); - usado para arredondar todas as bordas
2- sintaxe: border-radius: (superior esquerdo) (superior direito) (inferior direito) (inferior esquerdo); 
3- usado para arredondar cada borda com seu devido valor
4- sintaxe: border-radius: (valor - se e id) (valor - sd e ie);
5- primeiro valor = borda superior esquerda e inferior direita iguais - segundo valor = borda superior direita e inferior esquerda iguais
6- se sua div for quadrada (com width e height de mesmo valor) ao adicionar um border-radius: 50%; sua div ficará redonda

<-- Bordas decoradas -->

1- sintaxe: border-image-source: url("nome do arquivo");
            border-image-slice: (tamanho);
            border-image-repeat: (repeat - stretch - round - space);
2- o border-image-source define qual imagem será usada como borda
3- o border-image-slice define como a imagem será cortada
4- o border-repeat define como a imagem vai preencher a borda 
5- (repeat) = repete a imagem - (stretch) = estica a imagem - (round) = repete ajustando o tamanho pra encaixar certinho - (space) = repete com espaçamento entre as partes
6- shorthand: border-image: (source) (slice) (repeat)
7- para utilizar o border-image é obrigatorio ter a declaração de uma borda - recomendado usar cor transparent

<-- Variáveis em CSS -->

1- O nome do espaço onde declaramos nossas variáveis em CSS é o root
2- Usamos o seletor :root {} para declarar variáveis
3- Para declarar qualquer variável, antes do nome colocamos --
4- Ex: :root {
            --cor0: #c5ebd6;
            --cor1: #83e1ad;
            --cor2: #3ddc84;
            --cor3: #2fa866;
            --cor4: #1a5c37;
            --cor5: #063d1e;
        }
5- Para utilizar essas variáveis declaramos o valor como var(--(nome da variável));
6- Ex: background-color: var(--cor0);
